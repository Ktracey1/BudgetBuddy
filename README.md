# BudgetBuddy
ASE 220 Final Project - Frontend

**Abstract**
This frontend is a React-based user interface for a grocery budgeting and price comparison application. It allows users to create accounts, log in securely, and manage grocery lists with associated budgets. Within each list, users can add, edit, and remove grocery items while entering prices from different stores to compare costs.

This application communicates with a backend API to perform all data operations, ensuring that user data is securely stored and persists across sessions. The interface is designed to be responsive and user-friendly, supporting both desktop and mobile screen sizes.

Core features include authentication, list management, item tracking, price comparison, and budget-friendly monitoring. The frontend emphasizes usability and clarity, helping users understand their spending and make cost-effective grocery decisions.
